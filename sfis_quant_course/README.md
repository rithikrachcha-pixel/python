# SFIS Quant Finance Course
## Southampton Finance & Investment Society — Head of Quant Finance

A fully-coded, end-to-end quantitative finance curriculum covering quant dev,
quant research, and quant trading. Each module is self-contained with theory,
implementations, and exercises.

---

## Curriculum Map

| Module | Topic | Key Skills |
|--------|-------|------------|
| 01 | Mathematical Foundations | Probability, Stats, Linear Algebra, Calculus |
| 02 | Financial Mathematics | TVM, Fixed Income, Derivatives Pricing |
| 03 | Data Analysis & Engineering | pandas, yfinance, cleaning, feature engineering |
| 04 | Portfolio Theory | MPT, Efficient Frontier, Factor Models, CAPM |
| 05 | Derivatives & Options | Black-Scholes, Greeks, Volatility Surface |
| 06 | Quant Strategies | Momentum, Mean Reversion, Pairs, Stat Arb |
| 07 | Risk Management | VaR, CVaR, Stress Testing, Drawdown |
| 08 | Machine Learning in Finance | Regression, Classification, LSTM, Transformers |
| 09 | Backtesting Framework | Vectorised & Event-Driven, Performance Analytics |
| 10 | Execution & Market Microstructure | Order types, Slippage, TWAP/VWAP, Alphas |

## Setup

```bash
pip install -r requirements.txt
```

## Running a Module

```bash
python module_01_foundations/01_probability_statistics.py
python module_04_portfolio_theory/01_modern_portfolio_theory.py
# etc.
```

## Philosophy

This course is built for **practitioners**, not academics. Every concept is
immediately followed by working Python code you can run, modify, and deploy.
By the end you will be able to:

- Research and backtest systematic strategies from scratch
- Build a full risk management framework
- Apply ML to alpha generation
- Understand market microstructure and execution quality
